--- layout: page title: Configuring Dalliance --- There are many options you can configure when creating a Browser object.
coordSystem
This defines the reference genome coordinate system for the browser. It's principally used when querying the DAS registry to find relevant datasources. It is also needed to correctly re-map data between coordinate systems. A typical configuration looks like: {%highlight javascript %} coordSystem: { speciesName: 'Danio rerio', taxon: 7955, auth: 'Zv', version: 9 } {%endhighlight%} The speciesName field can be a human-readable label, and isn't used when querying the registry. taxon should be an integer identified from the NCBI Taxonomy auth and version should, as far as possible, match authority and version strings used in the DAS Registry
