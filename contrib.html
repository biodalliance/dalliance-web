--- layout: page title: Contributing ---

Biodalliance is open source software under a BSD-style license. Contributions of code, documentation, bug reports, and suggestions are all very welcome.

Development happens in our GitHub repository. For smaller code contributions and bug fixes, sending a Github pull request is probably the easiest way to get your code merged. For larger contributions, it is worth contacting us via the mailing list or issue tracker to discuss how best to integrate your code.

If your contribution is going to add dependencies on external libraries, please propose and discuss this on the mailing list before starting work.
